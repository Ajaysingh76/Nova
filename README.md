# Nova

Nova is an open-source terminal coding agent for developers.

It helps you inspect codebases, edit files, run commands, use tools, and work through coding tasks from a fast terminal-first interface.

## Status

Nova started from the MIT-licensed OpenCode codebase and is now being developed independently. The project is currently being reworked and rebranded.

## Goals

- Fast terminal-native coding workflow
- Multi-model support
- Tool use with approval controls
- Patch-based code editing
- Session memory and project context
